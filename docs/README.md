# Нормативные акты (PDF)

Кладите сюда PDF-файлы нормативных документов. Они автоматически появятся в
разделе **«Нормативные акты»** приложения.

- Имя файла без расширения `.pdf` — это название документа для отображения.
- Подчёркивания в имени файла заменяются на пробелы.
  Например, `Приказ_29н.pdf` → «Приказ 29н».
- Папка сканируется при каждом запросе, перезапуск не нужен.

Этот файл (`README.md`) разделом игнорируется — учитываются только `*.pdf`.
Он нужен лишь для того, чтобы пустая папка `docs/` сохранялась в репозитории
и копировалась в Docker-образ.
